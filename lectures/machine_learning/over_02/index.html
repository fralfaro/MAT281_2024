    Overfitting II¬∂

    Para mitigar el overfitting, podemos aplicar varias t√©cnicas, entre ellas:

    • Recolectar m√°s datos: Ayuda al modelo a generalizar mejor y reducir el ajuste al ruido.
    • Regularizaci√≥n: Introducir una penalizaci√≥n a la complejidad del modelo mediante t√©cnicas de regularizaci√≥n.
    • Modelos ensamblados: Combinar varios modelos para mejorar la robustez y precisi√≥n.
    • Validaci√≥n cruzada: Evaluar el modelo en m√∫ltiples particiones para obtener una mejor estimaci√≥n de su desempe√±o.
    • Optimizaci√≥n de hiperpar√°metros: Ajustar par√°metros con t√©cnicas como grid search para encontrar el equilibrio √≥ptimo entre ajuste y generalizaci√≥n.
    • Reducci√≥n de la dimensi√≥n: Simplificar los datos para disminuir la complejidad del modelo.
    • Selecci√≥n de atributos: Elegir solo las caracter√≠sticas m√°s relevantes para mejorar la generalizaci√≥n.

    A continuaci√≥n, exploraremos ejemplos de algunos de estos m√©todos para reducir el sobreajuste.

    Validaci√≥n Cruzada¬∂

    La validaci√≥n cruzada consiste en dividir el conjunto de datos en $k$ particiones (o pliegues), t√≠picamente entre 5 y 10.

    En cada una de las $k$ iteraciones, se selecciona un pliegue diferente como conjunto de evaluaci√≥n, mientras que los otros $k-1$ pliegues se combinan para formar el conjunto de entrenamiento. Esto implica que, en cada iteraci√≥n, $(k-1) / k$ de los datos se utiliza para entrenamiento y $1 / k$ para evaluaci√≥n.

    Cada iteraci√≥n genera una estimaci√≥n del rendimiento del modelo, proporcionando una medida de su capacidad de generalizaci√≥n. Al finalizar, cada ejemplo ha sido utilizado una vez para evaluaci√≥n y $k-1$ veces para entrenamiento, permitiendo calcular la media y desviaci√≥n est√°ndar de la precisi√≥n, ofreciendo as√≠ una evaluaci√≥n robusta del modelo.

    Veamos un ejemplo en python, ocupando el conjunto de datos make_classification.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.model_selection import train_test_split, StratifiedKFold
from sklearn.datasets import make_classification
from sklearn.tree import DecisionTreeClassifier
import random

# Configuraci√≥n de semilla y gr√°ficos
random.seed(1982)
%matplotlib inline
sns.set_palette("deep", desat=0.6)
sns.set_context(rc={"figure.figsize": (12, 4)})

# Generaci√≥n de datos simulados
X, y = make_classification(n_samples=10000, n_features=100, n_informative=3, n_classes=2, random_state=1982)

# Divisi√≥n de datos en entrenamiento y evaluaci√≥n
x_train, x_eval, y_train, y_eval = train_test_split(X, y, test_size=0.35, random_state=1982)

# Definici√≥n del modelo
model = DecisionTreeClassifier(criterion='entropy', max_depth=5)

# Configuraci√≥n de validaci√≥n cruzada con 20 pliegues estratificados
skf = StratifiedKFold(n_splits=20)
precision_scores = []

# Validaci√≥n cruzada
for k, (train_index, test_index) in enumerate(skf.split(x_train, y_train)):
    X_train_fold, X_test_fold = x_train[train_index], x_train[test_index]
    y_train_fold, y_test_fold = y_train[train_index], y_train[test_index]
    
    # Entrenamiento y evaluaci√≥n en el pliegue actual
    model.fit(X_train_fold, y_train_fold)
    score = model.score(X_test_fold, y_test_fold)
    precision_scores.append(score)
    
    # Imprimir resultados del pliegue
    print(f'Pliegue: {k+1}, Distribuci√≥n de Clases: {np.bincount(y_train_fold)}, Precisi√≥n: {score:.3f}')

# Resultados generales
print("\nPrecisi√≥n media de la validaci√≥n cruzada:", np.mean(precision_scores))

import numpy as np
from sklearn.model_selection import cross_val_score, StratifiedKFold
from sklearn.datasets import make_classification
from sklearn.tree import DecisionTreeClassifier

# Generaci√≥n de datos simulados
X, y = make_classification(n_samples=10000, n_features=100, n_informative=3, n_classes=2, random_state=1982)

model = DecisionTreeClassifier(criterion='entropy', max_depth=5)

# Configuraci√≥n de validaci√≥n cruzada con 20 pliegues estratificados

# Uso de cross_val_score para obtener precisi√≥n en cada pliegue

# Resultados

print(f'Desviaci√≥n est√°ndar de la precisi√≥n: {np.std(scores):.3f}')

    M√°s Datos y Curvas de Aprendizaje¬∂

    • A veces, reducir el overfitting puede lograrse simplemente con m√°s datos. ¬°Dame m√°s datos y te predecir√© el futuro!
    • Sin embargo, en la pr√°ctica, conseguir datos adicionales no suele ser tan sencillo.
    • Una t√©cnica √∫til para entender y reducir el overfitting son las curvas de aprendizaje, que muestran c√≥mo var√≠a la precisi√≥n en funci√≥n del tama√±o del conjunto de datos de entrenamiento.

    Para graficar las curvas de aprendizaje es necesario ocupar el comando de sklearn llamado learning_curve.

    # Librer√≠as necesarias
import numpy as np
import matplotlib.pyplot as plt
from sklearn.model_selection import learning_curve
from sklearn.datasets import make_classification
from sklearn.tree import DecisionTreeClassifier
import seaborn as sns

# Configuraci√≥n de estilo de gr√°ficos
sns.set(style="whitegrid")
plt.figure(figsize=(10, 6))

# Generar un conjunto de datos simulado
X, y = make_classification(n_samples=2000, n_features=20, n_informative=15, random_state=42)

# Definir el modelo de clasificaci√≥n
model = DecisionTreeClassifier(max_depth=5, random_state=42)

# Calcular las curvas de aprendizaje
train_sizes, train_scores, test_scores = learning_curve(
    model, X, y, cv=5, scoring='accuracy', train_sizes=np.linspace(0.1, 1.0, 10), random_state=42
)

# Calcular media y desviaci√≥n est√°ndar de las precisiones en cada tama√±o de entrenamiento
train_mean = np.mean(train_scores, axis=1)
train_std = np.std(train_scores, axis=1)
test_mean = np.mean(test_scores, axis=1)
test_std = np.std(test_scores, axis=1)

# Graficar las curvas de aprendizaje
plt.plot(train_sizes, train_mean, 'o-', color="blue", label="Precisi√≥n en entrenamiento")
plt.fill_between(train_sizes, train_mean - train_std, train_mean + train_std, alpha=0.2, color="blue")

plt.plot(train_sizes, test_mean, 'o-', color="orange", label="Precisi√≥n en validaci√≥n")
plt.fill_between(train_sizes, test_mean - test_std, test_mean + test_std, alpha=0.2, color="orange")

# Configurar etiquetas y t√≠tulo del gr√°fico
plt.xlabel("Tama√±o del conjunto de entrenamiento")
plt.ylabel("Precisi√≥n")
plt.title("Curvas de Aprendizaje")
plt.legend(loc="best")
plt.show()

    Grid Search¬∂

    La optimizaci√≥n de par√°metros es clave para mejorar el rendimiento de los modelos de Machine Learning y reducir el overfitting. Una herramienta com√∫n para ello es grid search, que busca de manera exhaustiva la mejor combinaci√≥n de par√°metros para maximizar la precisi√≥n del modelo.

    Grid search prueba todas las combinaciones especificadas de valores de par√°metros y selecciona la que ofrece el mayor rendimiento, evaluando cada opci√≥n mediante validaci√≥n cruzada.

    # Importar librer√≠as necesarias
import numpy as np
from sklearn.model_selection import GridSearchCV
from sklearn.tree import DecisionTreeClassifier

# Definir el modelo base
model = DecisionTreeClassifier()

# Rango de par√°metros para Grid Search
param_grid = {
    'criterion': ['gini', 'entropy'],                    # Funci√≥n de evaluaci√≥n para la divisi√≥n de nodos
    'max_depth': [4, 5, 6, 7, 8, 9, 10, 11, 12, 15, 20, 30, 40, 50, 70, 90, 120, 150]  # Profundidades posibles
}

# Configuraci√≥n y aplicaci√≥n de Grid Search
gs = GridSearchCV(
    estimator=model, 
    param_grid=param_grid, 
    scoring='accuracy',    # M√©trica de evaluaci√≥n
    cv=5,                  # N√∫mero de particiones de validaci√≥n cruzada
    n_jobs=-1              # Usar todos los n√∫cleos disponibles para acelerar el proceso
)

# Entrenamiento de Grid Search
gs.fit(x_train, y_train)

# Resultados del mejor modelo
print("Mejor precisi√≥n en validaci√≥n cruzada:", gs.best_score_)
print("Mejores par√°metros:", gs.best_params_)

# Entrenar y evaluar el mejor modelo en el conjunto de evaluaci√≥n
mejor_modelo = gs.best_estimator_
mejor_modelo.fit(x_train, y_train)
print("Precisi√≥n en el conjunto de evaluaci√≥n: {:.3f}".format(mejor_modelo.score(x_eval, y_eval)))

    Reducci√≥n de Dimensionalidad¬∂

    La reducci√≥n de dimensionalidad es una t√©cnica √∫til para combatir el overfitting al simplificar el modelo eliminando caracter√≠sticas irrelevantes o redundantes. Esto reduce la complejidad del modelo y ayuda a que generalice mejor en datos nuevos.

    Una t√©cnica com√∫n para la reducci√≥n de dimensionalidad es An√°lisis de Componentes Principales (PCA), que transforma los datos originales en un nuevo espacio de menor dimensi√≥n manteniendo la mayor varianza posible. A continuaci√≥n, se muestra un ejemplo en Python usando PCA para reducir la dimensionalidad de un conjunto de datos y luego entrenar un modelo de clasificaci√≥n.

import numpy as np
from sklearn.decomposition import PCA
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
from sklearn.metrics import accuracy_score

# Generar datos simulados con 20 caracter√≠sticas
X, y = make_classification(n_samples=1000, n_features=20, n_informative=10, n_redundant=10, random_state=42)

# Dividir el conjunto de datos en entrenamiento y evaluaci√≥n
x_train, x_eval, y_train, y_eval = train_test_split(X, y, test_size=0.3, random_state=42)

# Aplicar PCA para reducir las caracter√≠sticas a 10 componentes principales
pca = PCA(n_components=10)
x_train_pca = pca.fit_transform(x_train)
x_eval_pca = pca.transform(x_eval)

# Definir y entrenar el modelo en los datos reducidos
model = DecisionTreeClassifier(max_depth=5, random_state=42)
model.fit(x_train_pca, y_train)

# Evaluar el modelo en los datos de evaluaci√≥n
y_pred = model.predict(x_eval_pca)
accuracy = accuracy_score(y_eval, y_pred)

print("Precisi√≥n con reducci√≥n de dimensionalidad (PCA): {:.3f}".format(accuracy))
